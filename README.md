# 个人网站

一个科技感十足的个人主页，展示我的技能和项目。

## 查看方式

直接在浏览器中打开 `index.html` 文件即可预览。

或者可以使用简单的 HTTP 服务器：

```bash
cd ~/longwei-dev
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 自定义

修改 `index.html` 中的内容：
- 联系邮箱：搜索 `your@email.com`
- 项目内容：修改 `#projects` 部分
- 技能：修改 `#skills` 部分
- 统计数据：修改 `.stats` 部分

# Azure Bay MVP Task Board (12 Weeks)

## Sprint 0（Week 1-2）需求冻结与原型
- [ ] 冻结MVP功能范围（角色互动/经营/4小游戏/图鉴）
- [ ] 确定8个核心页面信息架构
- [ ] Unity项目初始化（竖屏、资源分包、性能基线）
- [ ] Live2D导入验证（1个角色全参数）
- 验收标准：
  - 可运行竖屏包
  - 角色可触摸并触发3种表情

## Sprint 1（Week 3-4）互动核心
- [ ] 角色状态机（待机/互动/剧情）
- [ ] 好感与情绪系统（等级/经验/衰减）
- [ ] 送礼与日程系统（基础）
- [ ] 对话与剧情触发器
- 验收标准：
  - 1个角色可完整互动闭环
  - 好感升级触发剧情与动作解锁

## Sprint 2（Week 5-6）经营与离线
- [ ] 建筑系统（建造/升级/产出）
- [ ] 资源系统（贝壳/海星/材料）
- [ ] 离线收益系统（上限+倍率）
- [ ] 经营页面UI与结算弹窗
- 验收标准：
  - 建筑产出可结算
  - 离线收益上线可领取

## Sprint 3（Week 7-8）小游戏模块
- [ ] 钓鱼玩法（QTE版）
- [ ] 采摘玩法（连击计分）
- [ ] 拼图玩法（静态拼片）
- [ ] 节拍玩法（轻量判定）
- 验收标准：
  - 4玩法均可进入、游玩、结算
  - 奖励正确入账

## Sprint 4（Week 9-10）图鉴与长线
- [ ] 服装图鉴（解锁与预览）
- [ ] 道具图鉴（稀有度/来源）
- [ ] 剧情图鉴（章节进度）
- [ ] 成就系统（40条）
- 验收标准：
  - 图鉴进度可追踪
  - 解锁链路完整

## Sprint 5（Week 11）优化与兼容
- [ ] 模型分级加载
- [ ] 低端机降级策略（30fps/简化物理）
- [ ] 内存与发热优化
- [ ] 关键机型适配
- 验收标准：
  - 中端安卓稳定45-60fps
  - 无P0/P1阻断Bug

## Sprint 6（Week 12）联调与发布
- [ ] SDK接入（登录/支付/统计）
- [ ] 提审素材打包
- [ ] 首发活动配置
- [ ] 上线回滚预案
- 验收标准：
  - iOS/Android提审包可提交
  - 线上监控与热修流程可用

## 关键里程碑
- M1（W2）：互动Demo可玩
- M2（W6）：互动+经营闭环
- M3（W8）：四小游戏全部打通
- M4（W10）：图鉴与长线目标完成
- M5（W12）：可上线版本
